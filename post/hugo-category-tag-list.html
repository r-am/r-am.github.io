カテゴリーとタグの一覧

  • 2017/04/24
  • hugo

サイドバーなどにカテゴリーとタグの一覧を表示させたい時は.Site.Taxonomiesから拾ってくる。

カテゴリー

{{ if lt 0 (len .Site.Taxonomies.categories) }}
<ul>
	{{ range .Site.Taxonomies.categories.ByCount }}
	<li><a href="{{ $.Site.BaseURL | relURL }}categories/{{ .Name | urlize }}">{{ .Name }}<span>({{ .Count }})</span></a></li>
	{{ end }}
</ul>
{{ end }}

タグ

{{ if lt 0 (len .Site.Taxonomies.tags) }}
<ul>
	{{ range .Site.Taxonomies.tags.ByCount }}
	<li><a href="{{ $.Site.BaseURL | relURL }}tags/{{ .Name | urlize }}">{{ .Name }}<span>({{ .Count }})</span></a></li>
	{{ end }}
</ul>
{{ end }}

全部

{{ range $key, $value := .Site.Taxonomies }}
{{ if lt 0 (len $value.ByCount) }}
<ul>
	<li>{{ $key }}
		<ul>
			{{ range $value.ByCount }}
			<li><a href="{{ $.Site.BaseURL | relURL }}{{ $key }}/{{ .Name | urlize }}">{{ .Name }}</a></li>
			{{ end }}
		</ul>
	</li>
</ul>
{{ end }}
{{ end }}
